# График курса валют

График курса валют - это веб-приложение, предназначенное для отслеживания курса рубля по отношению к доллару, евро и юаню.(Тестовое задание)

---

## Установка и настройка

Для установки и запуска приложения необходим Node.js и npm.

1. Клонируйте репозиторий:

```bash
git clone git@github.com:Rabbit1430/ExchangeRates.git
```

2. Установите зависимости:

```bash
npm install
```

3. Запустите приложение:

```bash
npm run dev
```

4. Перейдите по адресу `http://localhost:5173/` в вашем веб-браузере Google Chrome.

## Основные функции

- **Выбор валюты**: Вы можете выбрать одну или несколько валют, курс которых вы хотите посмотреть.
- **Выбор периода**: Пользователь может указать начальную и конечную дату для отображения данных о курсе в выбранном периоде. Просто нажмите на область даты и выберите нужную. **Важно выбрать дату начальную от нынешней даты(канечной), не больше 30 дней, так как api(https://github.com/fawazahmed0/exchange-api.) предоставленный для тестовго задания не показывает больше 30 дней данные. Можно протестировать от марта месяца(на апреле месяца еще не тестировал)** :smile:
- **Отображение графика**: После выбора валют и периода приложение отображает график изменения курса для выбранных валют.

## Технологии

- React.js + Typescript: для создания пользовательского интерфейса.
- Recharts: для построения графиков курса валют.
- Axios: для выполнения HTTP-запросов к API.
